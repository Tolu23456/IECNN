# IECNN

**Iterative Emergent Convergent Neural Network**

A conceptual research project exploring a novel neural network architecture designed as a potential foundation for Artificial General Intelligence (AGI).

## Running the App

```bash
python server.py
```

The app will be available at `http://localhost:5000`.

## Project Structure

- `index.html` — Web-based documentation viewer
- `server.py` — Simple HTTP server
- `iecnn_notes.txt` — Core concept notes
